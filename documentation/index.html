Import Android Project

Import the unzipped project to Eclipse:

 

  1. Select File Menu >> Import
  2. Select Existing Projects into Workplace and click next. 
  3. Click Browse to browse the unzipped project (Cabbooking_app->CabBookingHome), mark copy into workspace, then click OK . 
  4. Click Finish, you could see the project imported to Eclipse as shown below  

 

Add Libraries

We have used three standard libraries in this project. you can find these libraries in Cabbooking_app folder

  • appcompat_v7
  • google-play-services_lib
  • LibSlideMenu

Import these libraries in workspace.

  1. Create a new project to generate appcompbat library for your project Goto File>New>Android Application Project
  2. Provide any name to the project and click next ,next ,next  and finish.
  3. appcombat_v7 library is automatically generated in your workspace as shown in below images.
  4. Import google_play_services as well.
  5. Right click on project,Goto properties>Android>Add
  6. Select appcombat_v7 ,LibSlideMenu and google_play_services, Click Ok.
  7. Your project is successfully imported.

 

 

Used sdk and library detail

This application and all the libraries are build on android api level 19.

To change build verion to API Level 19

  1. Go on your project , right click on it and select properties
  2. Select android on left pane and check  android api level 19 (Android 4.4.2)
  3. Click apply and ok .
  4. Clean your project by clicking Project

click ok and run your app.

Change Google Maps API Key

To Change Google map Key

You have to Register a new Google Map Api Key ,  if you need help  visit #Click Here

  1. Once you get  New Key Goto Android Manifesr.xml
  2. goto application tag ,goto meta tag  and find "com.google.android.maps.v2.API_KEY"

Set up Web Admin Panel

In the Source Code you have downloaded, there is a folder named "cabbooking-PHP". This folder contains the source code for web admin panel.

To set up Wed Admin, you will required

  • PHP enabeled Server - (Linux server recommended)
  • MySQL database . version 5.3 or above
  • A total server space of  minimum 1 GB to host web admin files and user generated data

Set Up Database

  • To set up database, first create a mySQL databae on your server using phpmyadmin or any other tool.
  • In the "cabbooking-PHP" folder, there is a SQL file called cabbookingccv2.sql
  • Import this file in your newly created database through phpmyadmin import as shown in figure below.

Upload and Set UP PHP Admin

  1. Upload the contents of folder - cabbooking-PHP - to the root or any folder of your server via filezilla or any FTP client.
  2. Go to admin > includes > config.inc.php and eng_config.inc.php and open these two files in IDE
  3. Change the Connection string for database, root path and admin details for your APP as shown in figure below

 

 

4. Go to includes > config.php and open this file in IDE

5.Change the Connection string for database, and GOOGLE API KEY -

  • Database details should be same as in previous step and
  • GOOGLE MAP API KEY should Will be key for server application in google developer console
  • Please note that the folder profile_pic should have write permissions.
